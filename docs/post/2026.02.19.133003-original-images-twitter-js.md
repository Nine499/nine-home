# 提速获取推特原图的油猴脚本

在日常浏览推特的时候，经常会碰到这样的情况：点开图片看到的是压缩版本，想要原图还得多点几次、复制链接再打开。为了解决这个小麻烦，我写了一个小巧的油猴脚本（[pbs.twimg.com-orig.js](https://github.com/Nine499/49js/raw/refs/heads/master/pbs.twimg.com-orig.js)），它的作用就是：右键打开推特图片时，自动跳转到原图 URL。

## 为什么需要这个脚本？

推特为了节省带宽，会给图片加上一层压缩，右键「在新标签页中打开」得到的是带参数的 URL（比如 `format=jpg&name=small`），想切换到原图还得手动拼 `format=jpg&name=orig`。手动操作既耗时间又容易出错，所以我把这件事交给脚本自动完成。

## 脚本怎么做的？

脚本监听推特页面上的链接点击事件，如果发现目标是 `pbs.twimg.com/media/` 这种图片，就自动构造原图的 URL（把原本的参数替换成 `?format=jpg&name=orig`），然后直接打开它。实现逻辑简单、几行代码，不会干扰推特原有的 UI。

## 如何使用？

1. 安装油猴插件（Tampermonkey、Violentmonkey 等）。
2. 新建脚本，把 [pbs.twimg.com-orig.js](https://github.com/Nine499/49js/raw/refs/heads/master/pbs.twimg.com-orig.js) 的内容粘进去。
3. 保存后刷新推特页面，右键图片即可直接打开原图。

## 结语

这种小工具就是给自己解决一些重复操作，把时间花在更有价值的事情上。欢迎大家在 GitHub 上提 issue、改进脚本，或者 fork 一份根据自己习惯做出调整。
