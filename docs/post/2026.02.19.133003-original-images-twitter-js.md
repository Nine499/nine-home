# 一键获取 X/Twitter 原图的油猴脚本

日常浏览 X（Twitter）时，经常会遇到这种情况：点开图片后看到的是压缩版本，想看原图还要多点几次，甚至要复制链接再手动改参数。

为了解决这个小麻烦，我写了一个油猴脚本：[pbs.twimg.com-orig.js](https://github.com/Nine499/49js/raw/refs/heads/master/pbs.twimg.com-orig.js)。它的作用很简单：当你右键在新标签页打开推文图片时，自动跳转到原图 URL。

## 为什么需要这个脚本？

X 为了节省带宽，图片链接通常会带参数（如 `name=small`、`name=medium`）。
如果想看原图，一般需要手动改成 `name=orig`（有时还要处理 `format` 参数）。

这类重复操作既耗时，也容易改错，所以直接交给脚本自动完成更省心。

## 脚本做了什么？

脚本会监听页面中的链接行为。
当目标链接匹配 `pbs.twimg.com/media/` 图片地址时，它会自动重写参数并跳转到原图（`name=orig`）。

实现逻辑很轻量，不改 UI，也不会影响日常浏览体验。

## 如何使用？

1. 安装用户脚本扩展（Tampermonkey / Violentmonkey 等）。
2. 新建脚本，并粘贴 [pbs.twimg.com-orig.js](https://github.com/Nine499/49js/raw/refs/heads/master/pbs.twimg.com-orig.js) 的内容。
3. 保存后刷新 X 页面，右键图片在新标签页打开即可直达原图。
