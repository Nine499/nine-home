# 一键直达推特原图的油猴脚本

在日常浏览推特时，相信不少人会碰到这样一个小烦恼：打开图片后看到的是压缩后的版本，想要原图还得再点几次、复制链接再打开。最近我写了一个小巧的油猴脚本（[pbs.twimg.com-orig.js](https://github.com/Nine499/49js/raw/refs/heads/master/pbs.twimg.com-orig.js)），它的目标非常明确——右键打开推特图片时，自动跳转到原图。

### 为什么写这个脚本？

推特为了节省带宽，会给图片套上一层压缩，默认右键“在新标签页中打开”拿到的是一个带参数的 URL，在不同尺寸间切换还得靠手动拼 `format=jpg&name=orig` 这些参数。手工操作累且容易出错。于是，我想：能不能让浏览器自己判断并跳到真正的原图 URL？答案是可以的。

### 脚本做了什么？

这个脚本监听用户在推特上的链接点击行为，如果检测到目标是 `pbs.twimg.com/media/` 形式的图片，就自动构造原图的 URL（也就是把默认的 `format=jpg&name=small` 改成 `?format=jpg&name=orig`），再打开它。几行代码就能完成，不影响正常浏览，也不会干涉推特官方的 UI。

### 如何使用？

1. 安装油猴（Tampermonkey、Violentmonkey 等）。
2. 新建脚本，复制粘贴 [pbs.twimg.com-orig.js](https://github.com/Nine499/49js/raw/refs/heads/master/pbs.twimg.com-orig.js) 的内容。
3. 保存后刷新推特页面，右键图片就能直接得到原图了。

### 小结

这类小工具就是“为自己解决小问题”的典型例子。代码不多，但能节省不少重复操作时间。欢迎大家在 GitHub 上提 issue 或改进脚本，也可以直接 fork 一份自定义自己的行为。
