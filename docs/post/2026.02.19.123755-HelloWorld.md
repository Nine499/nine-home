# Hello World

这是我的第一篇文章。先从最经典的 `Hello World` 开始，分别用几种常见语言实现，作为一个简单的开场。

## Python

```python
print("Hello World")
```

## HTML + JavaScript

```html
<!DOCTYPE html>
<html>
  <body>
    <script>
      document.write("Hello World");
    </script>
  </body>
</html>
```

## Java

```java
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}
```

## C

```c
#include <stdio.h>

int main() {
    printf("Hello World\n");
    return 0;
}
```

## Go

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello World")
}
```
